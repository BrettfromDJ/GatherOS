Privacy policy

Last updated: May 22, 2026

The GatherOS browser extension does not collect, store, sell, or transmit personal information to GatherOS or any third party.

What the extension handles

When you right-click an image and choose “Save to GatherOS”, the extension passes the following data only to the GatherOS desktop application running locally on your own machine, via Chrome’s native messaging protocol:

  • The image URL you right-clicked.
  • The URL and title of the page you were viewing.

This data never leaves your computer. It is used solely by the GatherOS desktop application to fetch the image and store it in your local library, with the page URL preserved as a source breadcrumb.

What the extension stores

The extension itself does not retain any data between invocations. It does not write to chrome.storage, does not maintain a remote server, and does not include any analytics or telemetry SDKs.

Permissions in use

  • contextMenus — to register the “Save to GatherOS” right-click menu item.
  • nativeMessaging — to communicate with the GatherOS desktop application installed on the same machine.
  • notifications — to show a single confirmation toast after each save attempt.

Third parties

None. The extension does not transfer user data to any third party. It only talks to the GatherOS desktop application on the user’s own computer.

Changes to this policy

If this policy changes in any material way, the updated version will be published at this URL with a new “Last updated” date.
